ArchLinux食用指南

发表于 2023-02-02

关于

本篇由一个普通的以Arch GNU/Linux为主力系统的初中用户编写。
主要记录一下本人是如何折腾Arch GNU/Linux使其成为日常使用系统的过程。

安装

这部分其实网上已经有很多教程了，请自行查看。
Archlinux Wiki CN 安装指南
本教程默认你安装好了基本组件并配置好了pacman且掌握了linux基本命令和pacman的用法

使用

桌面环境和驱动

我本人使用的桌面环境是KDE，总之这是一款美观且开箱即用的桌面环境。
使用以下命令来安装：
$ sudo pacman -S plasma konsole xorg dolphin(也可以用nemo)
$ sudo pacman -S nvidia nvidia-settings nvidia-prime nvidia-utils opencl-nvidia lib32-nvidia-utils lib32-opencl-nvidia # N卡闭源驱动 So Nvidia,Fuck You！
$ sudo pacman -S xf86-video-intel # intel核显
$ sudo pacman -S x-server # N卡推荐
$ sudo pacman -S plasma-wayland-session plasma-wayland-protocols qt5-wayland qt6-wayland xorg-xwayland # 使用Wayland，N卡不推荐
如果你的电脑配置不太好的话也可以选择xfce,Lxde(勉强能用)甚至用i3wm和dwm都可以（这两不是桌面环境）。

AUR

推荐用清华大学的AUR镜像源。
AUR包管理使用paru或yay。
paru在安装AUR包前会默认让你看一遍PKGBUILD(了解这个包是如何构建的)以及它展示AUR包的顺序是倒序的。
新手建议使用yay。

使用AUR

请先通过上面的超链接帮助完成AUR源的配置。
随后在终端里运行命令：
$ sudo pacman -S archlinuxcn-keyring # 这个一定要先安装再去安装yay或paru！！！
$ sudo pacman -S yay(装paru就把yay换成paru就可以了)
然后就可以用 yay / paru (-S) 软件名的形式来查找（安装）AUR软件了。

软件

游戏

在终端输入以下命令来安装Steam
$ sudo pacman -S steam
Steam有一个基于wine的win游戏兼容层叫做Proton，可以在linux下运行大部分的windows游戏，具体请去protonDB查看
以及steam由于未知原因无法正常显示字体（包括英文）
